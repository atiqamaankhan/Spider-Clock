# 🕷️ Spider Clock  

A creative and interactive clock built using **HTML, CSS, and JavaScript**.  
This project showcases a unique *Spider-themed clock design* alongside a *standard clock* for comparison.

---

## 🚀 Features  

- 🕒 Real-time clock using JavaScript  
- 🕷️ Creative Spider Clock UI  
- 🎨 Smooth animations and styling  
- ⚡ Lightweight and fast  
- 📱 Responsive design
